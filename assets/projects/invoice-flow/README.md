# InvoiceFlow Project Images

Add screenshots of your project here.

## How to Add Images

1. Place your screenshots in this folder (`assets/projects/invoice-flow/`)
2. Supported formats: PNG, JPG, JPEG, WebP
3. Recommended size: 1200x800 pixels (16:10 aspect ratio)
4. Name files sequentially (e.g., `1.png`, `2.png`, `3.png`)

## Updating the Portfolio

After adding images, update `data/projects.js` (and `data/projects.json`):

```json
{
  "id": "invoice-flow",
  "images": [
    "./assets/projects/invoice-flow/1.png",
    "./assets/projects/invoice-flow/2.png"
  ]
}
```

The first image will be shown as the thumbnail on the project card.
